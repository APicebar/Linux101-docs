#进程、前后台、服务与例行性任务

            * 进程存在意义，基本属性（出现在htop上的条目），进程间通信方式。

        2. htop的使用（在进程调度方面的应用）
            * 进程间关系演示（进程树），信号传输实现（trap）<杀死几乎所有进程会如何 kill -9 -1>
            * *htop的其他选项（查看文件描述符），从而理解一些特殊文件的存在（如0号stdin，1号stdout，2号stderr）
        3. 将无需交互的程序放入后台
            * ctrl + Z挂起，bg与fg，nohup过继进程
        4. 守护进程与服务
            * 守护进程的工作模式，htop中守护进程的举例。
            * *配置ssh服务演示
        5. 例行性任务设置
            * 定时任务，比如定时upgrade。
